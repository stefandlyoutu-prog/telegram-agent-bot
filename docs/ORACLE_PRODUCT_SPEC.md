# m-Oracul — продуктовая спецификация

Источники: таблица-ТЗ (скрин), оферта `oferta_615108112390.docx`, текущий код `oracle_bot/`.

## Видение продукта

**m-Oracul** — Telegram-бот + Mini App + сайт для эзотерических/психологических разборов с воронкой монетизации.

| Канал | Роль |
|-------|------|
| **@MOracul_bot** | Основной продукт: 30+ практик, paywall, рефералка |
| **Mini App** | Каталог, быстрые кнопки, утренние пуши по теме |
| **Сайт /landing** | SEO, продажа, оферта, вход в бота |
| **Каналы** | Прогрев: M_Topgoroskop, signsvishe, auragirlss |
| **Admin CRM /admin** | Воронка, статистика, рассылки |
| **Пуши** | Реактивация + конверсия (лимит → реферал → премиум) |

## Тарифы (фаза 1 — без оплаты картой)

| Тариф | Условие |
|-------|---------|
| **Free** | 2 чтения/день на раздел |
| **Реферал** | +2 бонусных чтения за каждого друга |
| **10 друзей** | Безлимит на 365 дней |
| **Премиум** | 299 ₽/мес безлимит *(Stars — фаза 2)* |

## Воронка продаж

```
Трафик (каналы / SEO / реф) → /start (+ оферта)
  → Первое чтение (teaser бесплатно)
  → 🔒 Deep / лимит
  → Реферал ИЛИ Премиум (позже)
  → Retention: streak, пуши, карта дня
```

## Что УЖЕ есть в коде

- 30+ модулей, LLM, palm vision, funnel teaser/lock
- Stars + реферальный paywall A/B
- Пуши: welcome, inactive, limit, referral_nudge, morning topic
- `/stats`, `/daily`, `/broadcast`, REST `/api/admin/*`
- Mini App с каталогом и action → чат
- Каналы autopost, UTM `src_`, deep links `mod_`, `ref`
- Admin notify: новый юзер, возврат 7+ дней
